Communicatie en netwerken

Netwerk

Er zijn verschillende soorten netwerken. Dit zijn LAN, WAN en internet. Een netwerk is een systeem voor communicatie tussen twee of meerdere apparaten. Dit kan zowel draadloos als via een internetkabel.

LAN staat voor Local Aria Network. Dit is het netwerk thuis of op kantoor en komt daar niet buiten. Het is dus je eigen persoonlijke netwerk.

WAN staat voor Wide Aria Network en is het netwerk buiten je lokale omgeving. Dit wordt dus gebruikt als het over grotere afstanden gaat. Dit kan zelfs over een heel continent.

Het internet is een wereldwijd openbaar netwerk van computernetwerken.

Draadloze communicatie

Draadloze communicatie is een manier van communicatie via een netwerk wat niet via een draad gaat. Dit kan via twee manieren gaan, namelijk via WIFI en via mobiel netwerk.

WIFI is een certificatielabel voor producten voor draadloze netwerken. Dit wordt voor onder andere communicatie gebruikt. WIFI kan zowel via frequentiebanden en kanalen gaan. Bij frequentiebanden kan de WIFI over twee banden gaan. Namelijk 2.4 GHz en 5GHz. Dit zijn beide banden waarover de WIFI zonder licentie over uitgezonden kan worden. De ander optie zijn kanalen. Je moet zorgen dat je niet met te veel netwerken op een kanaal zit, anders zorgt dit namelijk voor storingen.

Om te bellen en te internetten met een mobiel buiten de WIFI is mobiel netwerk nodig. Er zijn verschillende generaties van mobiel netwerk. Momenteel zijn 4G en 5G de bekendste. De snelheid van 4G is 100 tot 1000 megabit per seconde. 5G zou in Nederland tot 800 gigabit per seconde moeten kunnen faciliteren. Deze gaan ook beide via een frequentieband, net zoals dit bij WIFI mogelijk is.

Hardware

Bij hardware horen alle fysieke componenten die in een computer een rol spelen. Hieronder staan een aantal componenten van hardware beschreven.

Switch: een switch is een apparaatje dat computers/elektronische apparaten met elkaar verbindt door ontvangen frames te ontvangen, te verwerken en weer door te sturen.

Router: een router is een apparaat dat minimaal twee computernetwerken met elkaar verbindt en data van het ene naar het andere netwerk verzendt. In een router zit een gateway. Dit is een punt in het netwerk dat als toegang naar een ander netwerk geldt.

Load balancer: load balancing is een techniek waarbij je werk over verschillende computers kan verdelen in een netwerk. Hierdoor zijn dingen te verdelen over verschillende processen, harde schijven en andere hulpbronnen.

Firewall: een firewall is een systeem dat de middelen van een netwerk of computer kan beschermen tegen misbruik van buitenaf. Hierdoor wordt bijvoorbeeld afluisteren lastiger gemaakt.

Modem: een modem maakt informatiesignalen geschikt om over een verbinding te transporteren naar andere apparaten. Dit gebeurt bijvoorbeeld bij een kabeltelefoon.

Access point: in een computernetwerk is het access point een apparaat dat het mogelijk maakt voor gebruikers om een verbinding te maken met een draadloos netwerk volgens een van de wifistandaarden. Dit is dus nodig om verbinding te maken met WIFI.

MAC-adres

MAC staat voor Media Access Control. Een MAC-adres is een bijna uniek identificatienummer dat aan een apparaat in een ethernetnetwerk is gekoppeld. Hierdoor is communicatie tussen verschillende apparaten mogelijk binnen een ethernetnetwerk.

IP-adres

IP in IP-adres staat voor Internet Protocol. Een IP-adres is een adres in een computernetwerk waarmee een netwerkkaart in een netwerk steeds gelijk geadresseerd kan worden binnen het IP-model. Elke computer die is aangesloten op het internet of netwerk heeft een IP-adresnummer waarmee hij zichtbaar is voor alle andere computers op het internet.

IPv4 is de vierde versie van IP. In IPv4 is een IP-adres een reeks van 32 bits. De adresruimte is daarom maximaal 232 = 4.294.967.296 IP-adressen.

De opvolger van IPv4 is IPv6. Dit is de zesde versie. In IPv6 is een IP-adres een reeks van 128 bits. Hierin zijn dus veel meer IP-adressen mogelijk.

Een subnet is een deelnetwerk is een netwerk, dat werkt via het Internet Protocol. Het wordt verdeelt over elkaar opeen volgende IP-adressen voor adressering op gescheiden fysieke netwerken.

DHCP

DHCP staat voor Dynamic Host Configuration Protocol. Dit is een computerprotocol waarin staat hoe een computer zijn netwerkinstellingen van een DHCP-server kan verkrijgen. Dit is gebaseerd op het Internet Protocol (IP).

DNS

DNS staat voor Domain Name System. Dit is het systeem en netwerkprotocol dat op het internet gebruikt wordt om namen van computers naar IP-adressen te vertalen en omgekeerd.

TCP/IP

TCP/IP is een reeks netwerkprotocollen die gebruikt worden voor het grootste deel van de netwerkcommunicatie tussen verschillende computers. Het internet is de bekendste vorm.

HTTP

HTTP staat voor HyperText Transfer Protocol. Dit is een protocol voor de communicatie tussen webgebruiker en webserver. Hierin staat welke vragen een gebruiker kan stellen en wat voor antwoorden de webserver hierop teruggeeft. Bij elke vraag wordt je naar een aparte URL getoond met het antwoord op de gestelde vraag.

HTTPS

HTTPS staat voor HyperText Transfer Protocol Secure. Dit is een uitbereiding van het HTTP-protocol wat zorgt voor veilige uitwisseling van gegevens. Gegevens die verstuurd worden, worden versleuteld, waardoor een buitenstaander geen toegang heeft tot de verstuurde gegevens. Dit wordt veel gebruikt bij betaaltransacties.
